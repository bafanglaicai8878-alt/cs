单独添加游戏

将 AppID 加入可入库清单，即可在游戏库搜索并生成 CDK

参数说明
· appid（必填）：Steam 主游戏 AppID，须为数字
· name（可选）：显示名称，留空则从 Steam 自动拉取
· 探测入库：检查 GitHub / Sudama 是否有清单（推荐开启）
· 强制加入：探测失败也写入清单（仅方便发卡，客户激活可能失败）
· 生成 lua：勾选后在服务端 Steam/config/stplug-in 真正入库（需 config.json 里 Custom_Steam_Path + GitHub Token + 清单源有该游戏）
DLC / 升级包请填主游戏 AppID，不要填 DLC 编号。
